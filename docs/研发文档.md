## 2. 状态管理方案

### 2.1 全局状态 (App globalData)
```typescript
interface GlobalData {
  salaryConfig: SalaryConfig    // 薪资配置
  derivedRates: DerivedRates    // 派生费率 (时薪/分薪/秒薪)
  currentState: WorkState       // 当前状态机状态
  todayEarnings: TodayEarnings  // 今日收益数据
  isOnboarded: boolean          // 是否完成引导
}
```

### 2.2 EventBus 事件
| 事件名 | 触发时机 | 数据 |
|--------|---------|------|
| `salary:updated` | 薪资配置变更 | `SalaryConfig` |
| `state:changed` | 状态机切换 | `{ from, to }` |
| `tick` | 每 100ms | `{ earnings, state }` |
| `data:cleared` | 清空全局数据 | 无 |

### 2.3 数据持久化键
| Key | 类型 | 说明 |
|-----|------|------|
| `SALARY_CONFIG` | `SalaryConfig` | 薪资配置 |
| `TODAY_EARNINGS` | `TodayEarnings` | 今日累计收益 |
| `MONTH_EARNINGS` | `number` | 本月累计 |
| `IS_ONBOARDED` | `boolean` | 引导完成标记 |
| `THEME` | `string` | 主题色 |
| `STATE_LOG` | `StateLog[]` | 今日状态切换记录 |

## 3. 核心组件拆解

### 3.1 计算引擎 (Engine)
```
输入: SalaryConfig { monthlySalary, workDaysPerMonth, hoursPerDay }
输出: DerivedRates { hourlyRate, minuteRate, secondRate }

公式:
  hourlyRate = monthlySalary / (workDaysPerMonth × hoursPerDay)
  minuteRate = hourlyRate / 60
  secondRate = minuteRate / 60

每 100ms 增量: secondRate × 0.1
白嫖模式增量: secondRate × 0.1 × 1.2
```

### 3.2 状态机 (StateMachine)
```
States: working | slacking | charging | sleeping

Transitions:
  working → slacking  (手动, 仅工作时间)
  slacking → working  (手动, 仅工作时间)
  working → charging  (自动, 到达下班时间)
  charging → working  (自动, 到达上班时间)
  * → sleeping        (自动, 00:00-05:00)
  sleeping → charging (自动, 05:00)

Guards:
  - 非工作时间禁止切换至 charging: Toast 提示
  - 非搬砖中禁止切换至 slacking: Toast 提示
  - sleeping 时间段强制锁定
```

### 3.3 热力图 (Heatmap)
```
总长 = 24 小时 = 100%
各色块占比实时计算:
  搬砖 = 实际搬砖时长 / 24h
  白嫖 = 实际白嫖时长 / 24h
  充电 = 实际充电时长 / 24h
  修仙 = 实际修仙时长 / 24h
约束: 四项之和恒等于 100%
```

### 3.4 倒计时模块
```
离下班 = endTime - now
环形进度 = (now - startTime) / (endTime - startTime)
边缘 Case: 已下班时文案切换，进度固定 100%

发薪倒计 = payDay - today (跨月处理)
周末进度 = 基于 workDayMode 计算
```

## 4. 接口约定

### 4.1 SalaryConfig 类型
```typescript
interface SalaryConfig {
  monthlySalary: number       // 月薪 (元)
  workDaysPerMonth: number    // 月工作日
  hoursPerDay: number         // 日工时
  startTime: string           // 上班时间 "HH:mm"
  endTime: string             // 下班时间 "HH:mm"
  lunchBreakHours: number     // 午休时长 (小时)
  workDayMode: 'double' | 'alternate' | 'single'  // 双休/大小周/单休
  payDay: number              // 发薪日 (1-31)
}
```

### 4.2 DerivedRates 类型
```typescript
interface DerivedRates {
  hourlyRate: number
  minuteRate: number
  secondRate: number
}
```

### 4.3 WorkState 类型
```typescript
type WorkState = 'working' | 'slacking' | 'charging' | 'sleeping'
```

### 4.4 TodayEarnings 类型
```typescript
interface TodayEarnings {
  date: string                // "YYYY-MM-DD"
  totalEarnings: number       // 今日总额
  workEarnings: number        // 搬砖所得
  slackEarnings: number       // 白嫖收益
  workDuration: number        // 搬砖时长 (秒)
  slackDuration: number       // 白嫖时长 (秒)
  chargeDuration: number      // 充电时长 (秒)
  sleepDuration: number       // 修仙时长 (秒)
}
```
