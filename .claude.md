# 纯利时刻 (PureProfit) - 项目上下文

## 项目概述
微信小程序，实时追踪打工人每秒收益。基于用户设定的薪资配置，可视化搬砖收益、白嫖收益等。

## 技术栈
- 微信小程序原生开发 (TypeScript + WXML + WXSS)
- TypeScript 严格模式 (`strict: true`, `noImplicitAny`, `strictNullChecks`)
- Skyline 渲染引擎已启用
- 不使用任何 UI 框架，纯原生实现

## 核心业务逻辑

### 计算引擎
- 时薪 = 月薪 / (月工作日 × 日工时)
- 分薪 = 时薪 / 60
- 秒薪 = 分薪 / 60
- 每 100ms 刷新: DisplayAmount += (秒薪 × 0.1)

### 状态机 (4 种互斥状态)
| 状态 | 条件 | 收益 |
|------|------|------|
| 搬砖中 | 工作时间自动激活 | 100% 秒薪 |
| 白嫖中 | 搬砖中手动切换 | 1.2x 秒薪 |
| 充电中 | 非工作时间 | 0 或自定义被动收益 |
| 修仙中 | 00:00-05:00 | 0 |

### 数据持久化
- wx.setStorageSync / wx.getStorageSync
- 页面销毁前同步增量至 Storage

## UI 视觉要点
- 品牌色: 能量金 #FFB800, 深空黑 #1A1D2E, 生机绿 #00C38B
- 字体: PingFang SC (中文), DIN Alternate / SF Pro Display (数字)
- 圆角: 外层容器 32px, 按钮/标签 12px 或胶囊
- 投影: 多层复合阴影，禁止单层阴影
- 图标: Lucide/Phosphor 线描风格
- 禁止: 纯黑 #000, 纯白边框, 锐利直角, 彩虹渐变
