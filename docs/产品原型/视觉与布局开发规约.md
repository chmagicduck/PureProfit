# 视觉与布局开发规约

## 1. 品牌定义与资源处理 (Brand & Assets)
- **应用名称:** 纯利时刻 (PureProfit)
- **品牌色:** 能量金 (#FFB800)、深空黑 (#1A1D2E)、生机绿 (#00C38B)。
- **美术资源占位:** - 所有未提供切图的 Icon/插画，请使用 `view` 绘制纯色占位块。
  - 占位块需标注：`[Asset Placeholder: Description]`。
  - 必须使用 `Lucide` 或 `Phosphor` 线描风格 Icon。

## 2. 4步引导流程：新手村 (Onboarding Flow)

### Step 1: 欢迎页 (Welcome)
- **布局:** 顶部 60% 留给金币 3D 旋转动画占位；底部 40% 内容区。
- **元素:** - 标题: 「开启你的纯利时刻」 (Bold, 48rpx)
  - 描述: 「让每一秒搬砖都变得有据可循」 (rgba(0,0,0,0.4))
  - 按钮: [开始配置] (能量金背景，带微弱呼吸动效)

### Step 2: 填写月薪 (Salary Set)
- **布局:** 居中大输入框。
- **交互:** - 输入时，下方实时计算并跳动显示：`当前秒薪: ¥0.0042`。
  - 数字变动需带有 `countUp` 滚动效果，增加“爽感”。

### Step 3: 配置工时 (Work Config)
- **列表式选择器:** - 上下班时间 (TimePicker)
  - 午休时长 (Slider: 0-2h)
  - 工作日模式 (Pill Toggle: 周末双休/大小周/单休)
  - 发薪日 (Input: 1-31)

### Step 4: 完成激励 (Success)
- **布局:** 全屏金币雨粒子效果 (Canvas/Animation)。
- **核心显示:** 「配置成功！你的纯利引擎已启动」。
- **主题预览:** 提供 4 个色块 (金/绿/蓝/紫)，点击即时切换 UI 主色调预览。

## 4. 个人中心页布局 (Profile Page)

- **Header:** 用户头像 + 用户名 + ID (简约流线设计)。
- **核心功能区 (Menu List):**
  - **[Icon] 薪资设置:** 点击进入修改流程。
  - **[Icon] 提醒偏好:** 下班提醒、发薪提醒。
  - **[Icon] 主题装扮:** 切换 4 色主题。
- **危险操作区 (Bottom):**
  - **[Button: Ghost Style] 清空全局数据:** - 需二次弹窗确认。
    - 样式：浅灰色文字，无底色，仅在点击时高亮。

## 5. 交互约束
- **全局反馈:** 所有的卡片点击需触发 `Heavy` 震动反馈 (Taptic Feedback)。
- **进入动画:** 首页加载时，数字需从 0 滚动至当前数值，卡片逐个由下至上淡入 (Offset: 20px, Duration: 0.4s)。
